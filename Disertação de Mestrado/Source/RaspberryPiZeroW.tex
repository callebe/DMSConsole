
Raspberry Pi é uma serie de computadores de tamanho reduzido construídos em uma única placa de circuito impresso (\textit{single-board computers}). Desenvolvido pela Raspberry Pi Foundation, os modelos da série Raspberry Pi possuem um Broadcom SoC (\textit{system on a chip}, integrado com um processador ARM e uma unidade de processamento gráfico (GPU) on-board \cite{AdafruitR}. Mesmo sendo criado com o intuito de promover ensino de ciências da computação em escolas de ensino básico, o Raspberry Pi tem sido encontrado em diversas aplicações na bibliografia por conta de sua versatilidade, preço baixo e tamanho reduzido \cite{Tso},\cite{Priyanka}.

Em fevereiro de 2017, a Raspberry Pi Foundation lançou um novo modelo de Raspberry Pi no mercado, o Raspberry Pi Zero W visto na Figura (\ref{fig:RaspberryPiZeroW}). Este novo modelo possui a metade do tamanho do modelo Raspberry Pi 3 A+, porém com o dobro de recursos e conectividade Wi-Fi. O Pi Zero W possui um processador ARM de 1GHz, memoria RAM de 512 MB, porta USB OTG (\textit{On-The-Go}), porta Minie HDMI, conector para camera CSI e 40 pinos de GPIO (\textit{General Purpose Input/Output}) \cite{Adafruit}. 

O Raspberry Pi Zero W, assim como todos os modelos Raspberry Pi, suportam sistemas operacionais baseados no kernel Linux, dispondo de toda as bibliotecas básicas destes tipos de sistemas. Em especial a Raspberry Pi Foundation disponibiliza uma versão de Linux, baseada na distribuição Debian (\cite{debian}), chamada Raspbian. Raspbian foi desenvolvido com o intuito de otimizar os recursos computacionais disponíveis no hardware, permitindo rodar uma interface gráfica amigável e ainda manter um bom desempenho. Em sua versão inicial, em 2012, o Raspbian possuía mais de 35000 pacotes de aplicações otimizadas para os modelos Raspberry Pi. Porém como este é um sistema Linux, é possível instalar pacotes de outras distribuições apenas incluindo e instalando as devidas dependências \cite{debian}. O Raspbian também possui compilador para as linguagens como Java, C/C++ e especialmente Python \citeauthoronline{Adafruit}.

O Raspberry  além de oferecer um dispositivo poderoso com um sistema operacional Linux, ainda agrega as funcionalidades dos microcontroladores mais comuns em sistemas embarcados. Entre estas funcionalidades está o conjunto de pinos da GPIO (\textit{General Propose Input and Output}), o qual é uma das grandes responsáveis pela facilidade com que o Raspberry Pi Zero W se encaixe nas mais diversas aplicações.  O sistema operacional, no  Raspberry Pi Zero W, tem acesso a 28 portas digitais de entrada/saída da GPIO, por ondem é possível realizar nos pinos da GPIO processos de leitura e escrita de sinais digitais, PWM (\textit{Pulse-Width Modulation}) e estabelecer comunicação pelos protocolos UART, I2C e SPI.

A Tabela (\ref{tab:PinOutRaspberryPiZero}) apresenta um resumo dos recursos presentes na GPIO do Raspberry Pi Zero W. Nesta tabela as portas que possuem o identificador GPIOx representam um pino da GPIO que pode ser usada para a leitura ou escrita de dados digitais. Já as portas marcados com SPI, UART ou I2C são aquelas que suportam leitura e escrita de dados digitais, mas que podem ser configuradas exclusivamente como interfaces de comunicação. Existe ainda os pinos de PWM, os quais podem ser usados para a modulação, os pinos de alimentação 3.3v/5V e GND (Terra), e ainda os pinos reservados DNC (\textit{Do Not Connect}). É importante notar que no Raspberry Pi Zero W existe apenas 1 interface de comunicação UART disponível, porém no projeto alvo deste trabalho existe a necessidade de 2 interfaces UART. Para solver este problema será utilizado um recurso de software para tornar alguns das portas da GPIO hábeis para a comunicação UART.
